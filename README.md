## Commands I've Practiced 
- mkdir "name " : to make a new folder .
- cd : to change the directory (folder)
- cd .. : to  go back one folder .
- ls : it shows list of all files in the folder . 
- ls -a : shows all hidden file .
- git clone (add link) : to clone any repo.
- git status : to check the status of that repo .
- git add (add file name) :  to add a new file .
- git commit -m : to commit (change) file .
- git push Origin main : to push the repo from local system (your device) to remote (github).



## My GitHub Workflow

1. **Create / Clone repo** → `git clone <url>`
2. **Make changes** → Edit files in VS Code
3. **Stage changes** → `git add .` or `git add <filename>`
4. **Commit changes** → `git commit -m "your message"`
5. **Push to GitHub** → `git push origin main`
6. **Code is live on GitHub!** 😎
